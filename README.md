# E-commerce Product Page

**Aim:** To practice building a responsive product page with interactive features and design accuracy.

**Overview:** A product detail page showcasing a product image gallery, selectable options, and add-to-cart functionality, closely matching the provided design.

**Skills Strengthened:** Responsive Layout | Image Gallery Interaction | DOM Manipulation | CSS Grid & Flexbox

**Challenges & Solutions:**  
- *Challenge:* Implementing the image gallery with thumbnail navigation → *Solution:* Used JavaScript to dynamically update the main image when a thumbnail is clicked.  
- *Challenge:* Keeping layout responsive across mobile and desktop → *Solution:* Applied CSS Grid and Flexbox strategically, with media queries for breakpoints.

**Key Takeaways:** Gained practical experience translating a design mockup into a fully responsive, interactive product page while maintaining clean code structure.
